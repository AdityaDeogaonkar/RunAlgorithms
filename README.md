# Coding & DSA Learning Platform
https://runalgorithms.com

A React-based web application for mastering Data Structures and Algorithms.

## Features
- **Topic Catalog**: Explore major DSA topics like Arrays, Strings, Graphs, Trees, and DP.
- **Curated Problems**: Practice with hand-picked problems from LeetCode.
- **Resources**: Direct links to top-tier roadmaps (NeetCode, Striver).
- **Responsive Design**: Built with Bootstrap for a clean and mobile-friendly UI.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start the Development Server**:
    ```bash
    npm start
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure
- `src/data.js`: Centralized file to manage topics and questions. Add new content here.
- `src/components/`: Reusable UI components (Navigation, TopicCard).
- `src/pages/`: Page views (Home, TopicDetail).
